# NEKOTORU 
NekoToru is a school Gymnasie Arbete Project which includes CRUD + sessions and login. 
Main functions of this website include the ability to post pictures via file upload, create your own account/profile and delete it whenever you want. 
This website is local, however if it wasnt, it would be very insecure since HTTPS is not used and everything does not undergo a try/catch thingy for unexpected errors!!
